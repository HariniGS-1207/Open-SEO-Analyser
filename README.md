# Open-SEO-Analyser 🚀

A full-stack SEO Analysis tool built to scrape web pages, extract critical SEO metrics, and provide optimization insights via a clean web dashboard. 

## 🛠️ Tech Stack

- **Backend:** Python (Web Scraping & Data Processing)
- **Frontend:** HTML, CSS, JavaScript (Interactive Dashboard)
- **Database:** SQLite / Relational Database

## 📂 Project Structure

```text
├── backend/          # Python scraping scripts and API logic
├── frontend/         # Web dashboard interface (HTML/CSS/JS)
├── .gitignore        # Keeps the repository clean
└── README.md         # Project documentation

```

**Prerequisites**
Python 3.x
A modern web browser

1. **Installation & Setup**
- Clone the repository:git clone https://github.com/HariniGS-1207/Open-SEO-Analyser.git
- cd Open-SEO-Analyser

2. **Set up the Backend environment:**
   ```bash
   # Create a virtual environment
   python -m venv venv
   
   # Activate it (Windows)
   .\venv\Scripts\activate
   
   # Activate it (Mac/Linux)
   source venv/bin/activate
3. **Run the tool:**
   - Execute your main Python script (e.g., test_scrape.py or your primary backend app) to analyze a URL.
   - Open the frontend dashboard to view the generated data.
